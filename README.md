# Find Largest and Smallest Element in an Array

## Problem
Write a program to find the smallest and largest element in an array.

## Algorithm
1. Initialize min and max with the first element.
2. Traverse the array.
3. Update min and max accordingly.

## Sample Input
10 3 25 7 2

## Sample Output
Smallest element: 2  
Largest element: 25

## Time Complexity
O(n)
